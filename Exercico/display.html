Propriedade Display

A propriedade display é a mais importante do CSS para controlar layout. Ela define como os elementos se comportam no fluxo da página.

  • Block: Ocupa a linha toda.
  • Inline: Fica na mesma linha.
  • Inline-block: Híbrido dos dois.
  • None: Desaparece.
  • Span: O container inline genérico.

1. Display: Block

Elementos em bloco sempre começam em uma nova linha e ocupam toda a largura disponível (esticam da esquerda à direita).

Exemplos comuns: <div>, <p>, <h1>.

.caixa {
  display: block;
  /* Largura será 100% por padrão */
}

Note que eles não ficam lado a lado, mesmo sendo curtos.

2. Display: Inline

Elementos inline ocupam apenas a largura necessária pelo seu conteúdo. Eles não quebram linha.

Atenção: Em elementos inline, definir width, height, margin-top ou margin-bottom geralmente não funciona.

.texto {
  display: inline;
  background: green;
}

3. Display: Inline-Block

O "melhor dos dois mundos". Ele flui na mesma linha (como o inline), mas permite definir largura, altura e margens (como o block).

É muito usado para menus de navegação, galerias e botões.

.card {
  display: inline-block;
  width: 100px;
  height: 100px;
}

4. Display: None

Remove o elemento completamente do documento. É como se ele nunca tivesse existido no HTML. O espaço que ele ocupava deixa de existir.

.escondido {
  display: none;
}

5. A Tag <span>

O <span> é um container genérico inline.

Diferente da <div> (que é block), o span é usado para estilizar pedaços de texto ou agrupar elementos dentro de uma linha sem quebrar o fluxo.

<p>Preço: <span class="destaque">R$ 50,00</span></p>

/* CSS */
.destaque { color: red; }

Este é um texto normal, mas estas palavras estão dentro de um span estilizado.
