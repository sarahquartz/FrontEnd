1. Listas em HTML

Existem dois tipos principais de listas: Ordenadas (ol) e Não-ordenadas (ul).

<ul>
  <li>Item sem ordem</li>
  <li>Outro item</li>
</ul>

<ol>
  <li>Primeiro</li>
  <li>Segundo</li>
</ol>

2. Box Model (CSS)

Tudo no CSS é uma caixa. O "Box Model" é composto por 4 camadas:

  1. Content: O conteúdo real (texto/img).
  2. Padding: Espaço entre o conteúdo e a borda.
  3. Border: A linha ao redor.
  4. Margin: Espaço fora da borda (afasta outros elementos).

3. A Tag <div>

A <div> é um conteiner genérico de bloco. Ela não faz nada visualmente sozinha, mas é usada para agrupar elementos e estilizá-los com CSS.

<div style="background: #ccc; padding: 10px;">
  <h3>Um grupo</h3>
  <p>Texto dentro da div.</p>
</div>

Estes elementos estão agrupados.

4. Classes em CSS

Classes (.nome) permitem aplicar o mesmo estilo a vários elementos diferentes. Você define no CSS e aplica no HTML com o atributo class="".

/* CSS */
.sucesso { color: green; }
.alerta { color: red; }

<!-- HTML -->
<p class="sucesso">Deu certo!</p>
<p class="alerta">Erro!</p>

5. Pseudoclasses

Pseudoclasses definem um estado especial de um elemento. A mais comum é :hover, que ativa quando passamos o mouse por cima.

.botao:hover {
  background-color: red;
}

6. Links Absolutos e Relativos

Usamos a tag <a href="..."> para links.

  • Absoluto: URL completa. Leva para outro site.
    Ex: https://google.com
  • Relativo: Caminho local. Leva para um arquivo no seu projeto.
    Ex: /contato.html ou imagens/foto.jpg
